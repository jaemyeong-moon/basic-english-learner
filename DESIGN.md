# UI/UX 설계 문서

작성일: 2026-04-11
담당: designer

---

## 1. 디자인 컨셉

- **스타일**: 깔끔하고 친근한 교육용 UI
- **컬러 팔레트**: 인디고(#4F46E5) 기반 + 밝은 배경, 동사별 고유 색상 아이콘
- **폰트**: Geist Sans (본문), Geist Mono (영어 예문)
- **다크 모드**: 지원 (Tailwind dark: 클래스 활용)

## 2. 페이지 구성

### 2.1 공통 레이아웃
- **헤더**: 로고 + 네비게이션 (홈, 동사 학습, 퀴즈)
- **메인 영역**: 페이지별 콘텐츠
- **푸터**: 간단한 카피라이트
- **반응형**: 모바일 우선 설계 (sm → md → lg)

### 2.2 홈페이지 (`/`)
- 서비스 소개 히어로 섹션
- "기초 영어 동사 마스터" 타이틀
- 간단한 설명 텍스트
- 동사 빠른 접근 카드 그리드 (3x3)
- CTA 버튼: "학습 시작하기", "퀴즈 풀기"

### 2.3 동사 목록 (`/verbs`)
- 9개 동사 카드 그리드 (모바일 1열, 태블릿 2열, 데스크톱 3열)
- 각 카드: 동사명, 한국어 뜻, 간단 설명, 예문 수
- 카드 클릭 시 상세 페이지로 이동

### 2.4 동사 상세 (`/verbs/[verb]`)
- 동사 헤더: 동사명 + 뜻 + 설명
- 예문 리스트: 영어 예문 + 한국어 해석
- 핵심 표현 하이라이트 (볼드 + 컬러)
- 패턴 태그 표시
- 이전/다음 동사 네비게이션

### 2.5 퀴즈 (`/quiz`)
- 진행 바 (현재 문제 / 전체 문제)
- 문제 카드: 빈칸 채우기 또는 객관식
- 힌트 버튼 (한국어 뜻)
- 결과 화면: 점수, 정답률, 오답 리뷰

## 3. 컴포넌트 구조

| 컴포넌트 | 위치 | 설명 |
|----------|------|------|
| Header | `components/ui/Header.tsx` | 네비게이션 헤더 |
| Footer | `components/ui/Footer.tsx` | 카피라이트 푸터 |
| VerbCard | `components/VerbCard.tsx` | 동사 카드 (목록용) |
| ExampleSentence | `components/ExampleSentence.tsx` | 예문 표시 |
| QuizQuestion | `components/QuizQuestion.tsx` | 퀴즈 문제 |
| ProgressBar | `components/ui/ProgressBar.tsx` | 퀴즈 진행 바 |

## 4. 화면 흐름

```
홈(/) → 동사 학습(/verbs) → 동사 상세(/verbs/[verb])
홈(/) → 퀴즈(/quiz) → 결과 표시
동사 상세 → 이전/다음 동사
```
